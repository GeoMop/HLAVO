# Exploration of data resources

This directory contains exploratory scripts and source-data examples used before
data is promoted into one of the schema-owned ingestion paths.

The production-like CHMI station/Open-Meteo update procedure is documented in
`chmi_stations/README.md`. The ALADIN source format notes are documented in
`chmi_nwp_aladin/README.md`; the automatic ALADIN surface update path is owned
by `../scrapper/`.
