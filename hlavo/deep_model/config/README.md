## Config files

Basic info:
around_mine_...  - models clipped by the "domain_around_mine" layer from GIS
with_mine_...    - models clipped by the "domain_with_mine" layer from GIS 

...coarse...     - horizontal size of the cell is 100x100 m
...fine...       - horizontal size of the cell is 40x40 m

...longterm      - calculation for 10 years (10 "stress periods", each one year long)

with_mine_longterm.yaml - the most recent model, includes pumping well

with_mine_longterm_rise.yaml  - experiment, injection of water instead of pumping.
